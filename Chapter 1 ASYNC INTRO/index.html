A very simple example of Async JS

  • Async JS doesn't stop the execution of the statements if there is a statement which is time consuming.
  • Instead it keeps executing that statement out of the normal flow
  • It keeps executing the statements in the flow
  • It calls back a funciton when that time eating statement finishes its process.
  • Best example is setTimeOut function of JS. It calls a function back after the specified time without stopping the next instructions in sequence of the code
  • See the console
  • The call back function is invoked after the specified time and the rest of the statements are not stopped from execution before that.
  • setTimeout is the simplest example to understand ASYNC javascript
