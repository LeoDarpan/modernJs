Fetching Data using Fetch API

  • XML httprequest is one way of fetching data from some resource. But its a very detailed way.
  • Fetch API simplifies all the process. It returns a promise after accessing the resource.
  • To get the data that we are looking for, we need to use json() function, which again returns a promise. So it cannot be assigned to a variable like parse().
  • So the promise is chained together as shown.
  • As can be seen, this way of getting data is much more simplified as compared to the XML way with which we started.

Here is the script:

    
        //------------Fetch API--------------
        fetch("data/games.json").then((response) => {
            console.log("resolved", response);
            return response.json();
        }).then((data) => {
            console.log(data);
        }).catch((err) => {
            console.log("rejected", err);
        });
