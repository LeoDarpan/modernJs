Async and await

  • This concept lets the promises chain together in a much more compact and clean way.
  • Async keyword makes a function asynchronous which always returns a promise.
  • Await removes the need to write the then and catch statements and use callback functions to get the data.
  • Await returns the result of promise.
  • This further boils down the code to just two lines as can be seen.
  • However, the call to the async function returns a promise so a then and catch method needs to be associated with the call to the funciton.
